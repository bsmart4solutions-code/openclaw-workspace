# 常见问题回复模板

> 客服 Agent 使用的回复模板集合

---

## 价格询问回复

**触发词：** 价格、多少钱、费用、quote、报价

**回复：**
```
您好！感谢您的咨询 😊

我们的装修价格取决于以下几个因素：
- 装修面积（平方尺）
- 装修风格和设计要求
- 材料选择（标准/高端/进口）
- 装修项目范围（局部/全屋）

请问您的房子是：
1. 公寓/condo 还是排屋/独立式？
2. 大约多少平方尺？
3. 是全屋装修还是局部（如厨房/卧室）？

了解后我可以给您一个初步估价范围。
```
